app
--------------------------------------

==== ============================================ =======================
Verb          URI                                  描述
==== ============================================ =======================
GET  /api/yaoxiao/app                              获取app列表
==== ============================================ =======================

请求参数
+++++++++++++++++++++++++

============= ============================== ======== ===========
 属性名                描述                     必填   例子
============= ============================== ======== ===========
 total          当前页，从0开始                  否    0
 pageSize      分页大小                         否     10
 currentPage   总数据量                         否     100
============= ============================== ======== ===========

返回数据
+++++++++++++++++++++++++++++

::
{
  "name": "Zhang Hua",
  "url": "http://blog.csdn.net/quqi99"
}

List need begin with *
++++++++++++++++++++++

The following attributes are used in the request body:

* ``name``

  Human-readable name. Might not be unique. Optional.

* ``url``

  url value.
